Interest in IT

Programming and
coding

My interests in IT are how programs are run through an interface and how the program made has so many functions to use and how easily it is showed to the users. These interests in IT began when I used bought my laptop and used my phone a lot I realised how easy it was to use it, but didn’t really understand how it was possible to run these functions.
